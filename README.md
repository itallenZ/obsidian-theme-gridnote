# Gridnote

A clean, grid-textured Obsidian theme with blue headings and colorful editor marks. Inspired by [Juejin](https://juejin.cn/)'s smartblue editor theme.

[中文文档](README_CN.md)

![Dark Mode](dark.png)

## Features

- 📐 Subtle grid background texture
- 🔵 Blue headings with left-border accent on H2
- ✍️ Colorful syntax marks in editor (headings, quotes, links, code, bold)
- 🔴 Inline code with red text on light pink background
- 📊 GitHub-style tables with zebra striping
- 💬 Blockquotes with pink background and purple-gray border
- 🌗 Full light & dark mode support
- 📈 Graph view styling

## Installation

### From Obsidian Community Themes

1. Open **Settings → Appearance → Themes → Manage**
2. Search for **Gridnote**
3. Click **Install and use**

### Manual Installation

1. Download `theme.css` and `manifest.json` from the [latest release](https://github.com/itallenZ/obsidian-theme-gridnote/releases/latest)
2. Create folder: `YOUR_VAULT/.obsidian/themes/Gridnote/`
3. Place both files in that folder
4. Go to **Settings → Appearance → Themes** and select **Gridnote**
